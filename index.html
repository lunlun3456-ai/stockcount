📦 Stock Count System — Setup

Connect your Google Sheet backend once, then all staff can use the app

1

Create a Google Sheet

Go to sheets.google.com → click + Blank → rename it Stock Count DB

2

Open Apps Script

In the sheet: Extensions → Apps Script → delete all default code → paste the Code.gs file contents → Save (Ctrl+S)

3

Deploy as Web App

Click Deploy → New deployment → Web app

  • Execute as: Me
  • Who has access: Anyone
  • Click Deploy → Authorize → Allow → copy the URL
4

Paste your Web App URL
