One-time setup — connect to Google Sheets

1

Create a Google Sheet

Go to sheets.google.com → Blank → rename it Stock Count DB

2

Paste Code.gs & run setup

Extensions → Apps Script → delete all code → paste Code.gs → Save → select setupAllSheets → ▶ Run → Authorize

3

Deploy as Web App

  • Deploy → New deployment → Web app
  • Execute as: Me  |  Who has access: Anyone
  • Deploy → Authorize → Allow → copy /exec URL
4

Paste URL and connect
